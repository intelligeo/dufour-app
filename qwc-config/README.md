# QWC2 Configuration Directory

This directory contains configuration files for QWC2 (QGIS Web Client) integration.

## Structure

```
qwc-config/
├── tenantConfig.json       # Main QWC configuration
├── permissions.json        # Layer/feature permissions (future)
└── themes/                 # Generated theme configs
    ├── dufour.json        # Auto-generated from dufour.qgs
    └── tactical.json      # Auto-generated from tactical.qgs
```

## Theme Generation

Theme JSON files are automatically generated by the Dufour Middleware API when:
- A new QGIS project is published via `/api/projects` endpoint
- The QWC service reads the .qgs XML and creates QWC2-compatible configuration

## Manual Theme Configuration

To manually customize a theme, edit the corresponding JSON file in `themes/` directory.

### Theme Structure

```json
{
  "title": "Project Title",
  "name": "project_name",
  "url": "http://qgis-server:80/...",
  "mapCrs": "EPSG:3857",
  "extent": [minx, miny, maxx, maxy],
  "scales": [1000000, 500000, ...],
  "backgroundLayers": [...],
  "themeLayers": [...]
}
```

## Integration with Frontend

The Dufour frontend loads themes via:
- `GET /api/v1/themes` - List all themes
- `GET /api/v1/themes/{name}` - Get specific theme config

This allows dynamic project switching without hardcoding layer configurations.
